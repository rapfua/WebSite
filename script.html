Bernoulli Bond Percolation on \(\mathbb{Z}^d\)

Setup:

  • \(G = (\mathbb{Z}^d, E)\)
  • \(E = \{(x, y) \in \mathbb{Z}^d \times \mathbb{Z}^d : ||x - y||_1 = 1\}\)

Probabilistic framework \((\Omega, \mathcal{F}, \mathbb{P_p})\)

  • \(\mathbb{P}_p(\omega) = p^{\sum_{e \in E} \omega(e)} (1 - p)^{\sum_{e \in E} (1 - \omega(e))}\)
  • \(\Omega = \{0, 1\}^E\)

  • \(\mathcal{F}=\) product \(\sigma\)-algebra

  • \(\mathbb{P}_p = \text{Bernoulli}(p)^{\bigotimes E}\)

Properties:

  • Let \(\omega, \eta \in \{0, 1\}^E\) be two configurations. We define the following ordering:

\(\qquad \qquad \omega \leq \eta \quad \iff \quad \omega(e) \leq \eta(e) \quad \forall e \in E.\)

  • An event \(A \in \mathcal{F}\) is increasing if

    \(\qquad \omega \in A \quad \& \quad \omega \leq \eta \implies \eta \in A.\)

    • Examples: \(\{x \xleftrightarrow{} y\} \text{ } \& \text{ } \{|C_x| \geq 10\}\) are increasing events.

    • Remark : \(A, B\) increasing \(\implies A \cap B \text{ } \& \text{ } A \cup B\) increasing.

    • Nonexample: \(\{|C_0| = 10\}\) is not increasing.

  • A function \(f: \Omega \to \mathbb{R}\) is increasing if

    \(\qquad \omega \leq \eta \implies f(\omega) \leq f(\eta).\)

    • Example: \(f(\omega) = |C_0(\omega)|\) is an increasing function.

    • Connection: event \(A\) increasing \(\iff\) function \(\mathbb{1}_A\) increasing.

  • Les \((\Omega, \mathcal{A}, P)\) be a probability space.
    A map \(X: \Omega \to \mathbb{R}\) is a random variable if \(X^{-1}(B) \stackrel{\text{notation}}{=} \{X \in B\} \in \mathcal{A}\) for all Borel sets \(B \in \mathcal{B}(\mathbb{R})\).
    In other words: \(X\) must a measurable map from \((\Omega, \mathcal{A})\) to \((\mathbb{R}, \mathcal{B}(\mathbb{R}))\). Note that the probability measure is irrelevant here.

    Each random variable induces a probability measure on \(\mathbb{R}\) by \(P_X(B) = P(X \in B) = P(X^{-1}(B))\) called the law of \(X\).

  • Proposition:
    1. Let \(A \in \mathcal{F}\) be an increasing event, then
      \(\qquad p \mapsto \mathbb{P}_p(A)\) is non-decreasing.

    2. Let \(f: \Omega \to \mathbb{R}\) be a measurable, increasing, nonnegative, bounded function. Then
      \(\qquad p \mapsto \mathbb{E}_p[f]\) is non-decreasing.

    • Proof:
        1. \(\implies\) 1. by taking \(f = \mathbb{1}_A\).

      • Define \(X_p(e) = \mathbb{1}_{\{U_e \leq p\}}\) so that
        \(\qquad E[f(X_p)] = E[f(\mathbb{1}_{\{U_e \leq p\}})] = p \cdot f(1) + (1 - p) \cdot f(0)\)

      • \(P(X_p(e) = 1) = p\) and \(P(X_p(e) = 0) = 1 - p\) so that the law associated to the \(X_p\)s is the same as the one associated to \(\mathbb{P}_p\).

      • Thus, the expectation of \(f\) under the law of \(X_p\) is the same as the expectation of \(f\) under \(\mathbb{P}_p\). Indeed, the change of variable formula gives
        \(\qquad E_P[g(Y)] = \int_{\Omega} g \circ Y dP = \int_{\mathbb{R}} g d \mu_{\text{pushforward of P with respect to Y}} = \int_{\mathbb{R}} g dP_Y\), where

        \(\qquad P_Y(B) = P(Y \in B) = P(Y^{-1}(B))\) is the law of \(Y\).

        In our case, this gives

        \(\qquad E[f(X_p)] = \int_{\mathbb{R}} f dP_{X_p} \quad (g \to f, \text{ } Y \to X_p)\) which is equal to

      • In the context of the proof, we have \(Y = X_p\) and \(g = f\) so that
        \(\qquad E[f(X_p)] = E_P[f(X_p)] = E_{\mathbb{P}_p}[f] = E_p[f]\)

1.1 The \(G(n, \kappa)\) Model

  • A spanning tree is a sub-graph that contains all original nodes and is a tree.

  • \(X\) Separable metric space \(\iff \exists D \subset X\) dense s.t. \(\forall x \in X, \forall \varepsilon > 0, \exists d \in D\) for which \(d(x, d) < \varepsilon.\)

    • E.g. \(\mathbf{Q}\) is dense in \(\mathbf{R}.\)

    • At the heart of the proof is the archimedean property of \(\mathbf{R}\), by which one can always empty a bathtub using a glass.

  • \(\mathcal{B} (X)\), the Borel σ-algebra on a topological space \(X\) is the smallest σ-algebra that contains all the open sets of \(X\).

  • Let \(\mu\) be a measure on a Borel σ-algebra. For each Borel set \(B\):

    • B is a \(\mu\)-continuity set \(\iff \mu(\partial B) = 0.\)

  • \(\mathcal{S}\) separable metric space & equipped with a Borel probability measure \(\mu \implies (\mathcal{S}, \mu)\) is called a ground space.

  • The natural of the symmetric measurable kernel \(\kappa: \mathcal{S}\times \mathcal{S}\to \mathbf{R}_{\geq 0}\) measures the density of edges.

    • In this case, “symmetric” simply means that \(\forall u, v \in \mathcal{S}: \kappa(u, v) = \kappa(v, u).\)

  • A kernel \(\kappa\) on a ground space \((\mathcal{S}, \mu)\) is irreducible if:

    • \(A \subset \mathcal{S}\) & \(\kappa\equiv 0\) a.e on \(A \times (\mathcal{S}\setminus A) \implies \mu(A) = 0\) or \(\mu(\mathcal{S}\setminus A) = 0.\)

  • A kernel \(\kappa\) on a ground space \((\mathcal{S}, \mu)\) is quasi-irreducible if:

    • \(\exists\) a \(\mu\)-continuity set \(\mathcal{S}' \subset \mathcal{S}\) s.t.

      • \(\mu(\mathcal{S}') > 0,\)

      • \(\kappa|_{\mathcal{S}' \times \mathcal{S}'}\) is irreducible,

      • \(\kappa|_{\overline{\mathcal{S}' \times \mathcal{S}'}} \equiv 0.\)

  • Irreducibility of kernels insures that \(\exists\) a unique giant cluster in the supercritical regime.

  • type of each vertex \(\in \mathcal{S}\) separable metric space equipped with a Borel probability measure \(\mu.\)

  • \(\nu_n(S) := \frac{\#\{i: \: x_i \in \mathcal{S}\}}{n} \xrightarrow{p} \mu(\mathcal{S}), \forall \: \mu\text{-continuity } S \subset \mathcal{S}.\)

  • When all conditions are met, we get the vertex space \(\nu := (\mathcal{S}, \mu, (x_n)).\)

  • A sequence of kernels \((\kappa_n)\) on a vertex space \((\mathcal{S}, \mu, (x_n))\) is graphical with limit \(\boldsymbol{\kappa}\) s.t.

    • \(\kappa\in L^1(\mathcal{S}\times \mathcal{S}, \mu \times \mu),\)

    • \(\kappa\) is continuous a.e.,
